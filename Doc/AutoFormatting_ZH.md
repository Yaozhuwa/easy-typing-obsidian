# 文本自动格式化

Easy Typing 在输入时自动格式化文本，在不同文字类型和行内元素之间添加适当的空格，自动大写句首字母，并保护特殊文本模式不被破坏。

## 自动大写

自动将英文句首字母大写。大写在输入时触发——如果结果不符合预期，可以立即撤销。

## 不同文字类型之间自动加空格

插件会自动在不同文字类别的相邻字符之间插入空格。例如，中文和英文之间，或中文和数字之间。

### 支持的文字类别

| 类别 | 字符范围 |
|------|---------|
| 中文 | CJK 统一汉字及扩展 A (U+4E00–U+9FFF, U+3400–U+4DBF) |
| 日文 | 平假名 (U+3040–U+309F)、片假名 (U+30A0–U+30FF, U+31F0–U+31FF) |
| 韩文 | 韩文音节 (U+AC00–U+D7AF)、韩文字母 (U+1100–U+11FF, U+3130–U+318F) |
| 英文 | A-Z, a-z |
| 数字 | 0-9 |
| 俄文 | 西里尔字母 (U+0400–U+04FF) |
| CJK | 元类别：中文 + 日文 + 韩文 |

### 文字类型对配置

在**自动格式化**设置标签页中，可以配置哪些文字类型对之间需要自动加空格。默认配置：
- 中文 ↔ 英文
- 中文 ↔ 数字
- 数字 ↔ 英文

你可以添加或删除类型对，也可以使用 CJK 元类别同时匹配中文、日文和韩文。

### 自定义文字类别

你可以使用 Unicode 字符类模式定义自定义文字类别。例如，定义一个"希腊文"类别，模式为 `\u0391-\u03C9`，就可以在希腊文和其他文字之间自动加空格。

## 行内元素间距

插件将每一行分割为不同片段：**文本**、**行内代码**、**行内公式**、**链接**和**用户自定义正则区块**。对于每种行内元素，可以配置间距策略：

| 策略 | 行为 |
|------|------|
| **无间距** | 无间距要求 |
| **软空格** | 软空格——标点符号或显式空格都可以作为分隔符 |
| **严格空格** | 必须有真正的空格字符 |

默认设置：
- 行内代码：软空格
- 行内公式：软空格
- 链接：软空格（智能模式在特定上下文中跳过 wikilink）

## 前缀词典

逐字输入单词时，插件可能会过早插入空格——例如，在你要输入 `n8n` 时，在输入 `n` 后就插入了空格。前缀词典通过抑制已知前缀的空格插入来避免这个问题。

每行一个条目，支持两种格式：

- **字面词**：如 `n8n`——在逐字输入 `n`、`n8`、`n8n` 的过程中都不会插入空格（支持前缀匹配）
- **正则模式**：用 `/.../` 包裹——如 `/[23][dD]/` 匹配 `2d`、`3D` 等。注意：正则**不支持前缀匹配**，只在 token 完整匹配时才抑制空格

## 软空格符号

你可以配置额外的软空格分隔符号（除了内置的常见全角标点符号外）。左侧和右侧的软空格符号分别配置。

## 用户自定义正则表达式区块

你可以用正则表达式定义特殊文本模式。匹配这些模式的文本被视为原子块——不会被自动格式化拆分，并且你可以指定它的间距行为。

### 语法

正则区块配置中每行的格式：

```
<正则表达式>|<左侧策略><右侧策略>
```

间距策略符号：
- `-`：无间距要求
- `+`：严格空格
- `=`：软空格（不常用，通常选 `-` 或 `+`）

以 `//` 开头的行视为注释。

### 默认正则区块

插件包含以下默认正则区块：

| 模式 | 策略 | 用途 |
|------|------|------|
| `{{.*?}}` | `++` | Templater / Dataview 表达式 |
| `<.*?>` | `--` | HTML 标签 |
| `\[\!.*?\][-+]{0,1}` | `-+` | Obsidian Callout 头部 |
| `(file:///\|https?://\|...)...` | `--` | URL 链接 |
| `[a-zA-Z0-9_\-.]+@[a-zA-Z0-9_\-.]+` | `++` | 电子邮箱 |
| `(?<!#)#[\u4e00-\u9fa5\w-\/]+` | `++` | Obsidian 标签 |

## 默认不会格式化的区域

自动格式化不会应用于：
- 代码块（` ``` ` 或 `~~~` 围栏）
- 数学公式块（`$$...$$`）
- Frontmatter（`---` 之间的 YAML）

## 命令

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| 格式化全文 | `Mod+Shift+S` | 对整个文档应用格式化 |
| 格式化选中/当前行 | `Mod+Shift+L` | 格式化选中文本或当前行 |
| 切换自动格式化 | `Ctrl+Tab` | 启用/禁用自动格式化 |

## 文件排除

在**其他**设置标签页中，可以列出文件路径（每行一个），排除这些文件不进行自动格式化。如果文件路径包含列出的任意字符串，该文件的自动格式化将被禁用。
