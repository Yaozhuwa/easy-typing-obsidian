# 编辑增强

Easy Typing 提供多种智能编辑功能，在输入时自动生效。这些功能旨在减少按键次数、提升编辑流畅度，尤其适合在 Obsidian 中使用中日韩文字的用户。

> 💡 本页介绍的是**编辑行为增强**（快捷键、光标跳转、粘贴等）。如果你在找**自动配对、全角半角标点转换、符号快捷输入**等功能，请参阅[规则引擎 → 内置规则](./CustomRules_ZH.md#内置规则速览)。

## 增强代码块编辑

增强代码块内的编辑体验：

| 操作 | 行为 |
|------|------|
| **Cmd/Ctrl+A** | 选中代码块内的全部内容（而非整个文档） |
| **Tab** | 添加缩进（使用仓库配置的缩进样式） |
| **Shift+Tab** | 移除一级缩进 |
| **Backspace** | 增强的删除行为 |
| **粘贴** | 粘贴内容自动尊重代码块的缩进级别 |

---

## 智能退格键

增强退格键对空列表项和空引用行的处理：

| 场景 | 效果 |
|------|------|
| 在空列表项上（如 `- `） | 按退格键移除列表标记 |
| 在空引用行上（如 `> `） | 按退格键移除引用前缀 |

---

## Tab 跳出配对符号

按 **Tab** 键将光标跳出配对符号。当光标在配对符号内部且没有活跃的 tabstop 占位符时，按 Tab 会将光标移到闭合符号之后。

支持的符号对：

| 分类 | 符号 |
|------|------|
| 中日韩符号对 | `【】`、`（）`、`《》`、`""`、`''`、`「」`、`『』` |
| 标准符号对 | `()`、`[]`、`{}`、`<>`、`''`、`""` |
| Markdown 符号对 | `$$`、`$`、`**`、`*`、`__`、`_`、`==`、`~~`、`[[]]` |

## 增强 Mod+A

渐进式选择扩展（`Cmd/Ctrl+A`）：

| 按下次数 | 效果 |
|---------|------|
| 第 1 次 | 选中当前行 |
| 第 2 次 | 选中当前文本块 |
| 第 3 次 | 选中全文 |

> ⚠️ 此功能需要在**编辑增强**设置标签页中手动启用。

---

## 智能粘贴

在列表或引用块中粘贴文本时，插件自动为每一行添加对应的前缀：

| 上下文 | 自动处理 |
|--------|---------|
| 在列表中 | 每行粘贴内容获得列表缩进 |
| 在引用块中 | 每行粘贴内容获得 `> ` 前缀 |


---

## 严格换行

启用后，按 Enter 会自动输入两个换行符，实现 Markdown 段落分隔效果。适合习惯"一个回车一个段落"的用户。

在**其他**设置标签页中配置。

---

## 折叠标题回车

在折叠的标题行按回车时，不展开折叠内容，直接在下方添加同级标题行。方便快速搭建文档大纲。

在**其他**设置标签页中配置。

## 命令参考

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| 当前行后新建行 | `Mod+Enter` | 在下方插入新行并移动光标 |
| 切换注释 | `Mod+/` | 切换行注释（代码块内识别语言） |
| 选择代码块 | — | 扩展选区到当前代码/引用块 |
| 增强 Mod+A | `Mod+A` | 渐进式选择：行 → 块 → 全文（需在设置中启用） |
