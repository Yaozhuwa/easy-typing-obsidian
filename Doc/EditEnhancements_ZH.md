# 编辑增强

Easy Typing 提供多种智能编辑功能，在输入时自动生效。这些功能旨在减少按键次数、提升编辑流畅度，尤其适合在 Obsidian 中使用中日韩文字的用户。

## 增强代码块编辑

增强代码块内的编辑体验：

- **Cmd/Ctrl+A 选中**：在代码块内选中代码块全部内容（而非整个文档）
- **Tab 缩进**：在代码块内按 Tab 添加缩进（使用仓库配置的缩进样式）
- **Shift+Tab 取消缩进**：移除一级缩进
- **增强删除**：代码块内增强的删除行为
- **智能粘贴**：在代码块内粘贴时，粘贴内容会尊重代码块的缩进级别

## 智能退格键

增强退格键对空列表项和空引用行的处理：
- 在空列表项上（如 `- `），按退格键移除列表标记
- 在空引用行上（如 `> `），按退格键移除引用前缀

## Tab 跳出配对符号

按 **Tab** 键将光标跳出配对符号。支持多种符号对：

- 中日韩符号对：`【】`、`（）`、`《》`、`""`、`''`、`「」`、`『』`
- 标准符号对：`()`、`[]`、`{}`、`<>`、`''`、`""`
- Markdown 符号对：`$$`、`$`、`**`、`*`、`__`、`_`、`==`、`~~`、`[[]]`

当光标在配对符号内部且没有活跃的 tabstop 占位符时，按 Tab 会将光标移到闭合符号之后。

## 增强 Mod+A

渐进式选择扩展（`Cmd/Ctrl+A`）：
1. 第一次按下：选中当前行
2. 第二次按下：选中当前文本块
3. 第三次按下：选中全文

此功能需要在**编辑增强**设置标签页中启用。

## 智能粘贴

在列表或引用块中粘贴文本时，插件自动为每一行添加对应的前缀：
- 在列表中：每行粘贴内容获得列表缩进
- 在引用块中：每行粘贴内容获得 `> ` 前缀

如果不需要此行为，使用**不格式化粘贴**（`Mod+Shift+V`）。

## 内置转换规则

插件包含一组内置转换规则（通过**内置规则**设置标签页管理），处理常见的中日韩编辑模式：

### 自动配对（优先级 10）

输入中日韩开括号或引号时，自动插入对应的闭合符号：
- `【` → `【|】`、`（` → `（|）`、`《` → `《|》`
- `"` → `"|"`、`'` → `'|'`、`「` → `「|」`、`『` → `『|』`

如果光标右侧已存在闭合符号，输入时会跳过而不是重复插入。

删除中日韩开括号时，会同时删除配对的闭合括号。

### 全角 ↔ 半角转换

**半角转全角**（优先级 15）：在中日韩字符后，英文标点自动转换为对应的全角标点：
- `,` → `，`、`.` → `。`、`?` → `？`、`!` → `！`
- `:` → `：`、`;` → `；`、`(` → `（|）`

**全角转半角**（优先级 3）：连续输入两个相同的全角标点，转换为半角：
- `。。` → `.`、`！！` → `!`、`？？` → `?`
- `，，` → `,`、`：：` → `:`、`；；` → `;`
- `（（` → `()`、`《《` → `<`

### 符号转换（优先级 10）

- `··`（两个中文间隔号）→ `` `|` ``（行内代码）
- `` `· `` → 围栏代码块
- `￥￥` / `¥¥` / `$￥` / `$¥` → `$|$`（行内公式）
- `$$`（行尾）→ `$$\n|\n$$`（块级公式）
- 行首 `》` → `> `（引用块）
- 行首 `、` → `/`

### 快速删除配对（优先级 30）

按退格键快速删除配对结构：
- `$|$` → 同时删除两个 `$`
- `==|==` → 同时删除两个 `==`
- `$$\n|\n$$` → 删除整个块级公式
- 空代码块 → 删除整个代码块
- `[[链接]]` 或 `![[嵌入]]` → 删除整个 wikilink

### 选中文字包裹（优先级 40）

选中文字后，输入触发字符进行包裹：
- `【` → `[选中文字]`
- `￥` / `¥` → `$选中文字$`
- `"` / `"` → `"选中文字"` / `"选中文字"`
- `'` / `'` → `'选中文字'` / `'选中文字'`
- `《` → `《选中文字》`、`（` → `（选中文字）`

### 引用处理（优先级 50）

- 在引用块上下文中输入 `>` 或 `》` 会添加一级引用前缀 `> `
- 自动在 `>` 前缀后补空格

## 命令

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| 当前行后新建行 | `Mod+Enter` | 在下方插入新行并移动光标 |
| 切换注释 | `Mod+/` | 切换行注释（代码块内识别语言） |
| 选择代码块 | — | 扩展选区到当前代码/引用块 |
| 增强 Mod+A | `Mod+A` | 渐进式选择：行 → 块 → 全文（需在设置中启用） |
